For contributing guidelines see the [Project Liberty Contributing Guidelines](https://github.com/LibertyDSNP/meta/blob/main/CONTRIBUTING.md).

## Running Tests
- `js/schemas` folder
- `npm run tests`

### To generate dsnp schemas from mainnet
- inside `js/schemas/scripts`
- run `npm run generate`
- check the newly generated `data.ts` file inside `js/schemas`
